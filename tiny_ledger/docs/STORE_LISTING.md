# App Store / Google Play 上架说明草稿（TinyLedger）

## 一句话介绍

帮助孩子用「虚拟记账」练习储蓄、花钱、定小目标，并用极简方式认识「增值」概念（学习模拟，非真实投资）。

## 隐私与合规要点（草稿）

- 本产品为教育向工具，**不涉及真实资金充值/提现/证券交易**。
- 首版数据默认**本地存储**（设备内 SQLite），不上传云端（若未来启用云同步，应更新隐私政策与商店文案）。
- 不包含第三方广告 SDK；核心路径不展示外链营销内容。

## 截图建议（6 张以内）

1. 适龄说明页（体现「家长陪同」与虚拟说明）。
2. 首页余额 + 最近流水。
3. 记一笔支出（分类为儿童友好标签）。
4. 目标列表 + 进度条。
5. 增值页教育标注 + 领取按钮（强调模拟）。
6. 更多页：减弱动效开关 + 家长占位入口。

## 分级与标签（占位）

- 建议以商店当地政策为准；文案强调「教育 / 家庭 / 理财启蒙」。
