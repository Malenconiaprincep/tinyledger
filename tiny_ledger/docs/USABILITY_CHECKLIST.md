# 关键路径可用性自测清单（8–10 岁）

在真机或模拟器上逐项勾选（建议家长陪同）：

- [ ] 首次打开能读完适龄说明，并顺利进入首页。
- [ ] 首页「当前余额」数字足够大，孩子能一眼看懂。
- [ ] 「记一笔收入 / 记一笔支出」按钮误触成本：误点后可取消返回，不会造成不可逆后果。
- [ ] 支出金额大于余额时，提示语是否口语化、可理解。
- [ ] 流水列表能区分收入/支出/目标转入/学习奖励（模拟）。
- [ ] 目标页：创建目标、查看进度条、从余额转入、达成后出现鼓励反馈（非金钱奖励）。
- [ ] 增值页：固定教育标注始终可见；未到领取周期时提示是否清晰。
- [ ] 更多页：应用内「减弱动效」开关是否生效；系统「减少动画」开启时是否更安静。
- [ ] 家长占位页：是否明确「首版不收集敏感个人信息」。
